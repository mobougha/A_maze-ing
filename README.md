## Description

**A-Maze-ing** is a maze generator in Python. It reads a configuration file, generates a random maze (perfect or not), displays it interactively in the terminal, and writes the result to a file using a hexadecimal wall encoding.  
The project includes a reusable Python package `mazegen_bestteam` that can be installed via pip.
